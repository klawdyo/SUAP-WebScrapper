# SUAP WEB Scrapper

## Instalação

```sh
# 1) clone
git clone https://github.com/klawdyo/SUAP-WebScrapper.git

# 2) instale as dependências
yarn
# ou
npm install

# 3) Atualize o seu arquivo .env e o .env.test a partir do .env.example. Neste arquivo estão as configurações para o funcionamento do sistema

# 4) rode os testes para ver se tudo está funcionando.
yarn test

```
